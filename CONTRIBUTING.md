# Contributing

Thank you for taking an interest in Find+.

## Before opening an issue

Please check whether the problem is reproducible after a full editor reload (`Ctrl+F5`) and include the relevant versions of Node-RED, Node.js and the browser.

## Pull requests

Keep changes focused and avoid unrelated formatting rewrites. The plug-in intentionally has no runtime dependencies and should remain lightweight.

Before submitting a pull request, run:

```bash
npm run check
```

When changing the editor UI, verify both light and dark themes where possible.
